# Jullix

Integration for the Jullix Energy Management System (Innovoltus). Monitor solar, battery, grid, EV chargers, and smart plugs. Optional control of chargers and plugs. Supports both cloud API and local Jullix-Direct for real-time data.

## Features

- Real-time power: grid, solar, home, battery, captar
- Battery SoC and power
- Metering (electricity, gas)
- EV charger status and power; optional on/off control
- Smart plug power; optional on/off control
- Cost and savings sensors (optional)
- Jullix-Direct: local real-time data without internet
